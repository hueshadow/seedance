# seedance.free

The complete English-language resource for [Seedance](https://seedance.free) — ByteDance's AI video generation model.

**Live site: [seedance.free](https://seedance.free)**

## What's Inside

- **[Complete Guide](https://seedance.free/guide)** — all versions, platforms, and access methods
- **[Prompt Builder](https://seedance.free/tools/prompt-builder)** — interactive tool to construct Seedance prompts
- **[API Documentation](https://seedance.free/api)** — code examples for fal.ai, Replicate & Volcano Engine (JS/Python/cURL)
- **[Version Comparison](https://seedance.free/versions)** — 1.0 Lite/Mini/Pro/Fast → 1.5 Pro → 2.0
- **[Pricing & Free Methods](https://seedance.free/pricing)** — all platforms compared, free access options
- **[Registration Guide](https://seedance.free/register)** — how to sign up without a Chinese phone number
- **[Credit Calculator](https://seedance.free/tools/calculator)** — estimate costs across platforms
- **[Model Comparison](https://seedance.free/tools/compare)** — side-by-side AI video model comparison
- **[Blog](https://seedance.free/blog)** — tutorials, comparisons, and prompt guides
- **[FAQ](https://seedance.free/faq)** — common questions answered with structured data

## Tech Stack

- **Framework**: Next.js 16 (App Router) + TypeScript
- **Styling**: Tailwind CSS v4
- **Content**: MDX with `next-mdx-remote`
- **Fonts**: Space Grotesk (display) + DM Sans (body)
- **Deployment**: Cloudflare Pages
- **SEO**: JSON-LD structured data, static sitemap, Open Graph

## Getting Started

```bash
pnpm install
pnpm dev
```

Build for production:

```bash
pnpm build
```

Output is in `out/` (static export).
